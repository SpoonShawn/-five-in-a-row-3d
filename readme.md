### 运行环境

Unity 2019.4.32或其兼容版本。源Unity项目在unity项目.zip文件中，请将五子棋.exe与其他文件在同一目录下运行。

### 主要内容

* 3D五子棋对弈，支持悔棋等操作。

![error](https://github.com/SpoonShawn/-five-in-a-row-3d/blob/master/img/image-20250120233457399.png)
